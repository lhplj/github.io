求整数二进制位最右(低)1的位置

lhplj
# 对于任意整数N,求N的二进制位最右1的位置
int pos = N & ((~N) + 1);

对原数先取反再加1,最后再与原数进行与操作即可

使用场景

  1. 求整数N二进制中1的数量
    int count = 0;
int rightOne = -1;
while (N != 0) {
    # 找最后边为1的位置
    rightOne = N & ((~N) + 1);
    count++;
    # 将N中最后边的1消掉
    N ^= rightOne;
}
